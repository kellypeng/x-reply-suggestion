Reply ideas

One-time setup (5 min)

This page generates reply ideas for a tweet you share to it. Set up once, then it lives in your X share sheet.

1. Add your Anthropic API key

Tap top-right, paste your key, Save. Get one at console.anthropic.com.

2. Build the iOS Shortcut (3 actions)

Open the Shortcuts app on your iPhone, tap +, then:

  1. Tap the settings icon (top-right). Set Name to "X Reply", turn on Show in Share Sheet, set Share Sheet Types to URLs only.
  2. Search for and add URL Encode. The input is the Shortcut Input (URL passed from share sheet).
  3. Add Text. Tap into it and type:
    https://x-reply-helper.kellypeng.com/m.html?url=
    Then tap the magic variable bar above the keyboard and insert the URL Encoded variable at the end so it reads:
    https://...m.html?url=[URL Encoded]
  4. Add Open URLs. Set the URL field to the magic variable for the Text from step 3.
  5. Done. Tap the back arrow to save.
3. Try it
  1. Open the X app. Find any tweet you want to reply to.
  2. Tap the share icon → tap X Reply in the share sheet.
  3. Safari opens this page with the tweet pre-loaded. Tap a reply card → it copies to clipboard. Switch back to X, paste in the reply box.

Optional: type an angle in the box above the cards and tap Generate to rewrite the reply in your specific direction.
